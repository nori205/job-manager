初期設定

GoogleアカウントでログインするにはOAuthクライアントIDが必要です。

  1. Google Cloud Console でプロジェクト作成
  2. 「APIとサービス」→「Google Drive API」を有効化
  3. 「認証情報」→「OAuth 2.0 クライアントID」を作成
  4. アプリの種類：ウェブアプリケーション
  5. 「承認済みのJavaScriptオリジン」にこのアプリのURL
    （例: https://example.github.io）を追加
  6. クライアントIDをコピーして下に貼り付け

※ GitHub Pagesで無料公開するのが最も簡単です
